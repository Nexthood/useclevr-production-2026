# UseClevr Code Review Skill

## Description
Performs comprehensive code review of TypeScript, React, and Next.js codebases with focus on security, performance, and best practices.

## Capabilities
- Security vulnerability detection
- Performance optimization suggestions
- TypeScript type safety review
- React/Next.js best practices
- Code quality assessment

## Usage
Invoke this skill when reviewing code for:
- Security vulnerabilities
- Performance issues
- Code quality improvements
- Best practice compliance
- TypeScript errors

## Focus Areas

### Security
- SQL injection prevention
- XSS protection
- Authentication/authorization
- Input validation
- Secret handling

### Performance
- Database queries
- React re-renders
- Bundle size
- API response times

### Code Quality
- Error handling
- Type safety
- Component structure
- State management

## Output Format
Provides detailed review with:
- Issue description
- File location
- Severity level
- Suggested fix
