| Layer | Technology |
|-------|------------|
| Frontend | Next.js 14, React 18, Tailwind CSS |
| Backend | Next.js API Routes, NextAuth v5 |
| Database | Neon PostgreSQL (serverless) |
| ORM | Drizzle ORM |
| AI | Google Gemini, DeepSeek |
| Payments | Disabled |
| Storage | AWS S3 / Cloudflare R2 |
| Deployment | Railway, Nixpacks, pnpm |
