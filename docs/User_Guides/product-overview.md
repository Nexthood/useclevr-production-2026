# Product Overview

UseClevr helps users upload business datasets, review dashboards, ask AI questions, and download reports.

## Main Workflow

1. Sign in.
2. Upload a CSV or business dataset.
3. Review generated KPIs, charts, forecasts, and insights.
4. Ask questions in the AI assistant.
5. Download reports from the downloads area.

## Key Areas

| Area | Purpose |
| --- | --- |
| Dashboard | Shows KPIs, charts, and business summaries. |
| Upload | Adds new datasets for analysis. |
| Datasets | Lets users review uploaded data and analysis results. |
| Assistant | Answers dataset questions and explains verified results. |
| Downloads | Stores generated reports and files. |
| Settings | Holds account and product preferences. |
